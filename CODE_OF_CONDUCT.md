# Code of Conduct

Clio Coder welcomes focused, respectful technical collaboration.

Expected behavior:

- Be direct, factual, and constructive.
- Critique code and decisions, not people.
- Keep issues and pull requests on topic.
- Disclose conflicts of interest when they affect a proposal.
- Respect maintainer decisions on scope, roadmap, and release timing.

Unacceptable behavior:

- Harassment, threats, or personal attacks.
- Posting secrets, private logs, or credentials.
- Spam, low-effort automation, or generated noise.
- Bypassing review, CI, branch protection, or security policy.

The maintainer may edit, lock, close, hide, or ban activity that violates these
rules.
