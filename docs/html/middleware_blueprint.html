Clio Coder Middleware & Component Specifications

Component scanner

The scanner computes SHA-256 signatures of codebase blocks and compiles component records used in reviews to flag runtime behavior alterations.

Reload Classes

Reload Class Meaning
hot Can be hot-reloaded during active session runtime.
next-dispatch Applies to subsequent worker dispatches.
restart-required Requires full Clio restart to load schema updates safely.

Middleware Hooks and Effects

Middleware filters evaluate tool calls, turn lifecycle reminders, and compaction observation:

  • Hook IDs: before_tool, after_tool, turn_start, turn_end, on_compaction.
  • Effect Kinds: inject_reminder, annotate_tool_result, block_tool, protect_path, request_continuation.
