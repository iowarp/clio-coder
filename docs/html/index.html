Developer Quick Start

Run these commands to install Clio Coder locally from a source checkout and run a local doctor check:

# Clone the repository git clone https://github.com/iowarp/clio-coder.git cd clio-coder # Link CLI commands locally npm run install:local hash -r # Run diagnostics in your code repository cd /path/to/your/repo clio doctor --fix
