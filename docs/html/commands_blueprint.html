Operator Mode Safety Matrix

Active Mode Selection

Select an operator mode to simulate target evaluation and tool admission blockades from Clio Coder's safety model.

File Reads (e.g. view_file) Admitted
File Writes & Edits (e.g. replace_file_content) Admitted
Bash Exec (Curated Allowlist / Audited entries) Admitted
Bash Exec (Arbitrary / Privileged commands) Parks
Enforcement Domain: DEFAULT MODE
Default mode provides active write access and allows execution of safe and structured actions. Unallowlisted bash commands are parked and request Operator Elevation (Alt+S) to proceed.
Safety Policy Matrix Rules:
  • Admitted tools execute immediately.
  • Parked tools pop up the Super Mode approval screen in the TUI.
  • Operator must press Enter to elevate, or Esc to abort.
