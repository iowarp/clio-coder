Clio Coder Keybindings & Command List

App Keybindings

Alt bindings represent the main navigation channel. In macOS terminals, ensure Meta is set to Option.

Cycle thinking levels Shift+Tab
Open Session Tree Navigator Alt+T
Toggle Footer Dashboard size Alt+U
Open Model/Target Selector Alt+L
Cycle through scoped models Alt+J / Alt+K
Toggle Dispatch Board overlay Alt+W
Steer active stream from editor Enter
Queue after-run follow-up Alt+Enter

TUI Slash Commands

Command Usage Purpose
/quit /quit Exit Clio Coder
/help /help [query] Open interactive help center and key reference
/settings /settings Open interactive full-screen Settings Center
/targets /targets Manage target connections, health check, and live probes
/model /model [pattern] Open model selector or set a model from the live/catalog list
/skill /skill [name] [task] Open the Skills Hub or invoke a skill
/view /view [filter] | /view verify <runId> Browse artifacts and verify receipts
/run /run <agent> <task> Deploy worker agent locally
/context-view /context-view Visualize active context breakdown (system/files/history)
/resume /resume Resume a past session from history ledger
