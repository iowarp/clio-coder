Clio Coder Memory & Evidence Specifications

Evidence Corpus Layout

Evidence ID is derived deterministically from run/session/eval tasks and builds folder records under <dataDir>/evidence/:

  • overview.json: Stable summary metrics metadata.
  • transcript.md: Raw markdown chat segment log.
  • findings.json: Compiled findings and diagnostics list.

Memory Injection Rules

The TUI chat loop compiles approved memory constraints synchronously before a turn starts:

Constraint Default Limit Value
Allowed Scopes global, repo
Token Budget Max 400 estimated tokens
Max Record count Max 5 records
Required Status approved: true
Required Citation At least 1 linked Evidence Ref
