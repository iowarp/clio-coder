Clio Coder Platform Specifications

Platform Path Defaults

OS Config Directory Data Directory State Directory Cache Directory
Linux / Unix ~/.config/clio ~/.local/share/clio ~/.local/state/clio ~/.cache/clio
macOS ~/Library/Application Support/clio/config ~/Library/Application Support/clio/data ~/Library/Application Support/clio/state ~/Library/Caches/clio
Windows %APPDATA%\clio\config %APPDATA%\clio\data %LOCALAPPDATA%\clio\state %LOCALAPPDATA%\clio\cache

File & Permissions Matrix

File Name Role Permissions Removal Condition
settings.yaml Runtimes/Theme configurations 0o644 Uninstall Wiped
credentials.yaml API Keys/Tokens 0o600 (Strict owner rw) Uninstall Wiped
install.json Platform Installation version state 0o644 Uninstall Wiped
audit/*.jsonl Daily safety audit logs 0o644 Uninstall Wiped

Doctor Repair Semantics

clio doctor is read-only. clio doctor --fix creates missing structure, repairs credential permissions, refreshes install metadata, and rewrites settings.yaml only when known legacy settings keys are present. That repair path creates settings.yaml.bak before writing the current schema.
