Clio Coder Eval Runner Specifications

The local evaluation runner executes repository-local YAML task suites as deterministic subprocess checks. It is useful for comparing harness changes, prompts, tools, or local workflows without requiring the runner itself to call a model.

Task-file schema

Parsed by src/domains/eval/task-file.ts. Enforces relative cwd paths inside the task folder to prevent absolute escapes.

version: 1
tasks:
  - id: cli-json-smoke
    prompt: "Verify the CLI JSON mode still starts."
    cwd: fixtures/cli-json-smoke
    setup:
      - npm install
    verifier:
      - npm run build
      - node dist/cli/index.js --help
    timeoutMs: 60000

Failure Classes

Failure Class Meaning
setup_failed A setup command exited with a non-zero exit status code.
verifier_failed A verifier test command exited non-zero.
timeout A command exceeded its timeoutMs limit.
cwd_missing The resolved working directory directory did not exist.
